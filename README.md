# 🎓 EduPanel — Creator Dashboard

A modern, full-featured **educational platform dashboard** built for course creators, educators, and institutions. Manage courses, live classes, test series, mock tests, bundles, polls, and more — all from one clean, responsive interface.

</div>

---

## ✨ What This Project Does

EduPanel is a **creator-first dashboard** designed to give educators full control over their content, students, and performance — without the clutter. It brings together everything a modern online educator needs into one cohesive, beautifully designed panel.

### Core Features

- 📚 **Course Management** — Create, organize, and track all your courses in one place
- 🎥 **Live Classes** — Schedule and manage live sessions with ease
- 📝 **Mock Tests & Test Series** — Build and distribute assessments for your learners
- 📦 **Bundles** — Package courses together for better student value
- 📊 **Dashboard Analytics** — Enrollment activity, performance summaries, and creator shortcuts
- 🗳️ **Polls** — Engage your audience with interactive polls
- 🛤️ **Tracks** — Organize learning paths and student journeys
- 🔖 **Classification & Question Pool** — Structured content and question bank management
- 🧰 **Utilities & Code Tools** — Power-user tools built right in

## 🚀 Getting Started

### Prerequisites

Make sure you have **Node.js** installed on your system.  
Download it here → [nodejs.org](https://nodejs.org/en/)

### Installation

```bash
# Clone the repository
git clone https://github.com/wajiddaudtamboli/edupanel-dashboard

# Navigate into the project
cd edupanel-dashboard

# Install dependencies
npm install
```

### Running Locally

```bash
npm run dev
```

Your app will be live at → [http://localhost:5173](http://localhost:5173)

### Building for Production

```bash
npm run build
```
